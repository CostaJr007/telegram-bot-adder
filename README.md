# 🤖 Telegram Bot Adder

Tool for managing Telegram groups and automating member addition.

## ⚠️ Disclaimer

This tool is for **legitimate use only**. Always comply with Telegram's Terms of Service.

## 🚀 Features

- ✅ Automated member addition
- ✅ Group management tools
- ✅ User filtering
- ✅ Rate limiting compliance
- ✅ Activity logging

## 📋 Requirements

- Python 3.9+
- Telegram API credentials

## 🔧 Installation

 + "`" + @"
bash
git clone https://github.com/CostaJr007/botelegramadd.git
cd botelegramadd
pip install -r requirements.txt
 + "`" + @"

## 💡 Usage

Configure config.yaml with your API credentials:

 + "`" + @"
yaml
api_id: YOUR_API_ID
api_hash: YOUR_API_HASH
phone: YOUR_PHONE_NUMBER
 + "`" + @"

Run:

 + "`" + @"
bash
python main.py
 + "`" + @"
